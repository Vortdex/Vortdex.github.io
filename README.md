## 🛠️ Wie wurde das gebaut?
- **Design**: Mit [Lovable.app](https://lovable.dev) (Free Version)
- **Export**: Manuell über Browser-DevTools (Network-Tab → `index.html`, `style.css`, `assets/`)
- **Hosting**: Kostenlos auf [GitHub Pages](https://pages.github.com)

---

## 📁 Dateien
Alle Dateien wurden **manuell exportiert** – kein automatischer Export, kein Build-Prozess:
- `index.html`
- `style.css`
- `script.js`
- `assets/` (Bilder, Icons)

---

## 💡 Warum manuell?
- Keine Abhängigkeit von Lovable’s GitHub-Integration
- Volle Kontrolle über den Code
- Keine Build-Fehler, keine Workflows, keine `deploy.yml`
- Einfach hochladen → fertig
